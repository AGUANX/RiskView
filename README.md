📋 项目简介
RiskView 是一个面向多源监测场景的风险度量、场景划分与数据可视化分析平台。该项目针对地质灾害、森林火灾、河流水文、电网巡检等复杂监测场景，提供从风险量化评估、多场景自动划分到数据可视化展示的完整解决方案。
本系统基于空间统计与风险场理论，实现多维度风险指标计算、自适应场景聚类划分、以及交互式数据可视化，为无人机遥感监测与运筹优化决策提供数据支撑。
✨ 核心特性
- 🎯 多维度风险度量：支持灾害风险、安全风险、运营风险等多维度量化评估
- 🗺️ 自适应场景划分：基于风险分布特征的自动聚类与场景边界划分
- 📈 风险场建模：空间连续风险场构建与插值分析
- 🎨 多视图可视化：热力图、等值线、空间分布、时间序列等多种展示方式
- 🔄 多场景对比：支持不同场景下风险分布的对比分析
- 📦 数据接口标准化：统一的数据输入输出格式，易于集成
